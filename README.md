## General info

This project is simple posters of films. Live: https://simply-filmposters.netlify.app

## Technologies

Project is created with:

- HTML5
- CSS3
- Javascript
- React.js

## Setup

To run this project, install it locally using yarn:

```
$ cd ./movie-project
$ yarn add
$ yarn start
```
